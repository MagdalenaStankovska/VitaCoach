# 🏋️ VitaCoach — AI-Powered Personal Fitness Assistant

> A full-stack RAG (Retrieval-Augmented Generation) fitness platform powered by Gemini, ChromaDB, and Google Calendar — built to give users personalized workout plans, nutrition guidance, and smart scheduling tailored to their goals.

---

## 🧠 What is VitaCoach?

VitaCoach is an intelligent fitness assistant that combines the power of **Google Gemini**, **vector search**, and **real-world local data** to deliver a truly personalized health experience. Whether you want to build arm muscles or strengthen your legs, VitaCoach retrieves the most relevant exercises from its vector database, schedules your workouts, rest days, and meals directly on your Google Calendar, and even points you to nearby gyms and restaurants in Macedonia.

---

## ✨ Features

### 🤖 RAG-Powered Exercise Bot
- Chat with a **Gemini-powered assistant** and ask things like *"Give me an arm workout"* or *"What exercises target my legs?"*
- Exercises are stored and retrieved using **ChromaDB** (vector database), enabling semantic search over the full exercise library
- Each exercise response is enriched with a **YouTube video** fetched via the YouTube Data API for guided visual instructions

### 📅 Smart Google Calendar Integration
- Gemini analyzes your goals and automatically schedules:
  - 🏋️ **Workout sessions**
  - 🍽️ **Meal times**
  - 😴 **Rest & recovery days**
- Calendar events are created via **OAuth 2.0 token authentication** — no manual planning needed

### 🍽️ Nutrition & Restaurant Guidance
- Scraped real restaurant and food data from **[korpa.mk](https://korpa.mk)** — Macedonia's local food delivery platform
- Gemini recommends foods and nearby restaurants based on the **client's specific fitness goal** (bulking, cutting, maintaining, etc.)

### 🏟️ Nearby Gym Discovery
- Scraped gym listings from **[sportmaster.mk](https://sportmaster.mk)**
- VitaCoach surfaces relevant gyms near the user to make getting started as easy as possible

### 📋 Popular Training Programs
- Curated collection of **popular training programs** with full exercise breakdowns
- Each program is structured with sets, reps, rest times, and linked video demonstrations

### 🔐 Authentication
- Full **Login & Register** system
- Secure **OAuth 2.0** flow for Google Calendar access

## 🚀 Getting Started

### Prerequisites
- Python 3.10+
- Node.js 18+
- A Google Cloud project with **Gemini API**, **Google Calendar API**, and **YouTube Data API v3** enabled
- ChromaDB installed

### 1. Clone the repository
```bash
git clone https://github.com/your-username/vitacoach.git
cd vitacoach
```

### 2. Backend Setup
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

Create a `.env` file in `/backend`:
```env
GEMINI_API_KEY=your_gemini_api_key
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
YOUTUBE_API_KEY=your_youtube_api_key
SECRET_KEY=your_jwt_secret
```

Run the backend:
```bash
python main.py
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## 🔄 How It Works

```
User asks: "Give me a workout for bigger arms"
        │
        ▼
Gemini understands the goal (arm hypertrophy)
        │
        ▼
ChromaDB performs semantic search over exercise library
        │
        ▼
Top matching exercises are retrieved and passed to Gemini
        │
        ▼
Gemini generates a structured workout plan
        │
        ├──► YouTube API fetches demo videos for each exercise
        │
        ├──► Google Calendar creates workout, meal & rest events
        │
        └──► Nearby gyms (sportmaster.mk) & food recs (korpa.mk) are surfaced
```
