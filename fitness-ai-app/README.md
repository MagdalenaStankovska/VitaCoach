# 💚 VitaCoach AI - Your Smart Fitness Companion

A beautiful, modern fitness app powered by AI with an elegant glass morphism design.

## ✨ Features

### 🎨 Stunning Modern Design
- **Glass Morphism UI** - Beautiful frosted glass effect on cards and components
- **Gradient Backgrounds** - Animated gradients with radial color overlays
- **Smooth Animations** - Buttery smooth transitions and hover effects
- **Gradient Typography** - Eye-catching gradient text on headings
- **Fully Responsive** - Perfect on desktop, tablet, and mobile

### 🤖 AI-Powered Features
- Ask personalized fitness questions
- Get AI-generated workout plans
- Receive exercise recommendations with images
- Save favorite exercises to your personal plan
- Download workout plans as text files
- Watch tutorial videos for exercises

### 🌟 Beautiful UI Components
- **Sticky Navigation** - Glass navbar with elegant hover effects
- **Exercise Cards** - Interactive cards with image zoom on hover
- **Smooth Modals** - Slide-up modals with backdrop blur
- **Gradient Buttons** - Shimmer effects and smooth interactions
- **Custom Scrollbars** - Themed scrollbars with gradient colors

## 🎨 Design System

**Color Palette**
- Primary: Vibrant greens (#22c55e, #4ade80)
- Background: Deep blues (#0f172a, #1e1b4b)
- Text: Slate grays (#cbd5e1, #e2e8f0)

**Key Effects**
- Backdrop blur for glass morphism
- Gradient overlays and borders
- Smooth scale transforms on hover
- Animated shimmer effects
- Pulsing loader animations

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Start the backend (in separate terminal)
cd backend/fitness-rag-backend
python app.py
```

Visit `http://localhost:5173` to see your beautiful fitness app!

## 🛠️ Tech Stack

- **Frontend**: React 18 + Vite
- **Styling**: Modern CSS with advanced effects
- **Routing**: React Router v6
- **Backend**: Python Flask with RAG AI
- **UI/UX**: Glass morphism, gradients, animations

## 📱 Responsive Design

- 🖥️ Desktop: > 768px - Full layout with all effects
- 📱 Tablet: 481px - 768px - Optimized grid and spacing
- 📱 Mobile: < 480px - Single column, touch-friendly
